Easy Draw-

Easy Draw is a simple and handy Drawing tool developed using JavaFx which is used to Draw/Paint and Save the output file as (.png) Format in Local directory.

JavaFx Introduction

JavaFX is a set of graphics and media packages that enables developers to design, create, test, debug, and deploy rich client applications that operate consistently across diverse platforms. Learn More

How To Run :
    Download zip file or tar.gz from above link
  1. Unzip the file with winzip or any other unzip softwear
  2. Locate Easydraw.jar and double click on it The application will run on desktop
  3. Note :To run jar file you must have java runtime evironment (JVM)/Java(TM) Platform SE binary installed on System .
