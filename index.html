Easy Draw-

This is simple Draw on canvas application developed using JavaFx which is used to draw and save the output file as (.png) image in pc directory. How To Run : 1.To use this project open eclips and create a new javafx project.

  1. Name the project as DrawPad and copy paste the Draw2.java file in src /application pacakge .
  2. click on run as java application , aftre successful compilation the application will run on your desktop.
  3. to run jar file java runtime mevironment /Java(TM) Platform SE binary must installed on system .
